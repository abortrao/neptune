# Poseidon Specs

Filecoin's Poseidon specification document (as of Mar-01-2021) is published on [HackMD](https://hackmd.io/@jake/poseidon-spec). Additionally, [PDF](spec.pdf) and [Markdown](spec.md) versions are mirrored in this directory. To render the LaTeX in the Markdown file `spec.md`, the file's contents can be coppied into a new [HackMD](https://hackmd.io) note.
